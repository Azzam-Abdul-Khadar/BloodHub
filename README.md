## 🚀 1. Introduction

**BloodHub** is a full-stack MERN application that connects **blood donors** with **people in urgent need of blood**.

The goal of this project is to reduce emergency response time by providing a platform where users can:

- Register as donors  
- Search donors by blood group  
- Send blood requests  
- Manage authentication securely  

This project focuses on real-world impact, clean architecture, and scalable full-stack design.

## ▶️ 9. Running the Project Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Azzam-Abdul-Khadar/BloodHub.git
cd BloodHub
```

### 2️⃣ Install Dependencies

```bash
npm install
```

(If frontend and backend are separate folders)

```bash
cd backend
npm install

cd ../client
npm install
```

### 3️⃣ Configure Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4️⃣ Start the Application

```bash
npm run dev
```

Then open:

```
http://localhost:3000
```
